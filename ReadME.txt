This GUI based project was implemented in python 2 as a part of my 3rd semester academic curriculum. The Advance Contact Manager easily handles and stores your favorite and frequently used contacts . This project uses Tkinter, SQLite3, MessageBox libraies of python.

Just open the file named 'Project_Main_Page' in the folder 'Project_py'.
A Documentation will be opened automatically.

close this Documentation file and the project will appear on the screen by itself.
